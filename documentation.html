Project Overview

This project is a web-based implementation of LDU Matrix Decomposition, built as part of a college assignment. It provides an interactive calculator, theoretical explanation, and team information — all accessible from a single landing page.

Subject: Numerical Methods / Linear Algebra  |  Technology: HTML, CSS, JavaScript  |  Matrix size: Up to 8×8

Algorithm Used

The calculator uses Gaussian Elimination without pivoting to compute LDU decomposition of an n×n matrix A, such that:

A = L × D × U

Steps

1

Start with matrix A. Initialize L = I (identity), D = 0, U = I.

2

For each pivot position k, check that the pivot a[k][k] ≠ 0. Set D[k][k] = a[k][k].

3

Compute multipliers: L[i][k] = a[i][k] / a[k][k] for rows below the pivot, and eliminate entries below the pivot.

4

Set U[k][j] = a[k][j] / D[k][k] for columns to the right of the pivot.

5

Repeat until all pivots are processed. Verify by computing L × D × U and comparing with original A.

Complexity

OperationComplexity
LDU DecompositionO(n³)
Verification (L×D×U)O(n³)
Space (L, D, U matrices)O(n²)

How to Use the Calculator

1

Open LDU Factorization Calculator from the home page.

2

Enter the matrix dimension (between 1 and 8) in the input box.

3

Click "Generate Own Matrix" — a grid of input boxes will appear.

4

Fill in all matrix values (decimals are supported).

5

Click "Decompose" — L, D, and U matrices are displayed along with verification.

Limitations

The current implementation does not support pivoting, so matrices with a zero pivot (zero on the diagonal during elimination) will return an error message. Use a permuted version of such matrices.

Output Matrices Explained

MatrixColorDescription
L — Lower Triangular Blue Unit lower triangular matrix (diagonal = 1, zeros above diagonal)
D — Diagonal Green Diagonal matrix containing the pivot values from Gaussian elimination
U — Upper Triangular Yellow Unit upper triangular matrix (diagonal = 1, zeros below diagonal)
